# NovoTHOR Website

Sanity.io CMS managed site written in Next.js 14 App Router with TypeScript and Tailwind CSS for novothor.com

## Tech Stack

- Next.js 14 (App Router)
- TypeScript
- Tailwind CSS
- Sanity.io CMS
- Framer Motion
- Vercel

## Development

```bash
npm install
npm run dev
```

## CMS Access

Studio: `/studio`

## Environment Variables

```
NEXT_PUBLIC_SANITY_PROJECT_ID=69z2nvbz
NEXT_PUBLIC_SANITY_DATASET=production
```

## Deployment

Deployed on Vercel with automatic builds from main branch.

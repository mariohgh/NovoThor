# How to Edit Pages in CMS

## 1. Go to Studio
http://localhost:3000/studio

## 2. Create Pages

### Create Health Page
1. Click "Page" in sidebar
2. Click "Create new Page"
3. Title: `Health`
4. Slug: Click "Generate" → `health`
5. Content: Add text blocks
6. Click "Publish"

### Create Performance Page
1. Click "Page"
2. Click "Create new Page"
3. Title: `Performance`
4. Slug: `performance`
5. Add content
6. Publish

### Create Technology Page
Same steps with Title: `Technology`, Slug: `technology`

### Create About Page
Same steps with Title: `About`, Slug: `about`

### Create Locations Page
Same steps with Title: `Locations`, Slug: `locations`

## 3. Edit Home Page Content

1. Click "Home Page" in sidebar
2. Edit:
   - Hero title/subtitle
   - Sports section text
   - Military section text
3. Publish

## 4. View Changes

Go to http://localhost:3000 - changes appear instantly!

## What's Editable

✅ All page content
✅ Hero sections
✅ Sports/Military sections
✅ Benefits (add/edit)
✅ Team logos (add/edit)
